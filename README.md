# python-for-data-science


## Repository Structure

This repository is organized into two main sections:

### 1. `Theoretical_concepts/` folder
Contains the theoretical material for the course, divided into thematic units:

- **unit_0: Refermant conceptes**
	- Review and reinforcement of basic programming concepts in Python.
- **unit_1: Estructures de dades avançades**
	- Study of advanced data structures in Python (lists, dictionaries, sets, tuples, etc.).
- **unit_2: Ús avançat de funcions**
	- Advanced use of functions, arguments, lambda functions, decorators, and functional programming concepts.
- **unit_3: Fitxers i interacció amb el sistema**
	- Reading and writing files, path manipulation, and file system operations.
- **unit_4: Optimització, complexitat i profiling**
	- Code optimization, algorithmic complexity analysis, and profiling techniques.
- **unit_5: Optimització, paral·lelisme i concurrència**
	- Concurrent and parallel programming, introduction to threads and processes in Python.
- **unit_6: Testing, manteniment i desplegament**
	- Verification, testing, maintenance, and deployment of Python applications.

Each unit may contain data and practical examples in subfolders.

### 2. `PACs/` folder
Includes the solutions to the Continuous Assessment Tests (PACs) for the course.
